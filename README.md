        <div style="font-weight: 700; font-size: 15px; text-align: left; color: #282d33;"> 
        <ul dir="auto">
        <li> <strong>레디고</strong>는 IT 개발자들을 타깃으로 개발자와 기업 간 입사 지원부터 면접까지 취업 전반을 관리하고 화상 면접을 통해 채용의 모든 과정을 온라인으로 해결할 수 있도록 한 채용 연계 사이트입니다.</li>
        <li> 이력서와 자기소개서를 관리하여 <strong>취업 준비의 효율성</strong>을 높이고 프로필을 통해 지금까지 이뤄낸 자신만의 경력 사항에 대한 정보를 게재함으로써 입사 지원을 하지 않더라도 스카우트 기능을 통해 회원과 기업 간 채용의 기회가 단절되지 않도록 했습니다.</li>
        <li> 채용 외에도 외주 기능을 추가하여 프리랜서 개발자와 개인 또는 업체를 빠르게 연결하고, 다양한 프로젝트에 참여할 기회를 제공합니다.</li>
        <li> <strong>효율적인 채용 프로세스</strong>를 통해 채용 기간을 단축하고, 프로필과 이력서 관리를 통해 구직자가 자신의 경력을 체계적으로 정리할 수 있도록 지원합니다.</li>
        <li> 이를 통해 취업 준비 기간을 줄이고, 기업은 필요한 인재를 신속히 찾을 수 있어 <strong>업무 공백을 최소화할 수 있는 기대효과를 제공</strong>합니다.</li>    
        </ul>
